Unified Intent Schema (v2 — enriched)

intents table — 6 levels, full lifecycle, outcome tracking
── Identity ──
id string "int_<ts>_<rand>"
title string required
description text optional
slug string unique, auto-generated
source_fingerprint string unique (dedup across all ingest paths)

── Hierarchy ──
level integer 0-5
0=vision 1=goal 2=project 3=feature 4=task 5=execution
kind string goal|question|task|exploration|fix|audit|system
parent_id FK → intents (self-ref, cascade)
children_count integer cached counter (for tree rendering)

── Lifecycle (from Superman + Execution Router) ──
status string planned|active|researched|outlined|implementing|complete|blocked|archived
phase integer 1-5 (research→outline→implement→review→harvest)
clarity float 0.0-1.0 (from IntentFolder)
energy float 0.0-1.0 (from IntentFolder)
completion_pct integer 0-100 (computed from child status propagation)
↑ Superman pattern: all children complete → 100%, some → proportional, none → 0%

── Outcome Tracking (from OpenClaw) ──
priority integer 0-4 (P0=bypass, P1=<1min... P4=idle)
fitness_score float 0.0-1.0 (success rate of executions on this intent)
what_worked text JSON array (outcome journal)
what_failed text JSON array (outcome journal)
retry_count integer default 0 (circuit breaker tracking)

── Linkage ──
project_id FK → projects
goal_id FK → goals
task_id FK → tasks
proposal_id FK → proposals
execution_id FK → executions
brain_dump_item_id FK → inbox_items
harvested_intent_id FK → harvested_intents
session_id FK → claude_sessions

── Content + Context ──
wiki_path string vault note path
intent_folder_path string .superman/intents/<slug>
embedding binary vector (for clustering + similarity)
tags text JSON array
metadata text JSON map (extensible)
linked_code text JSON array of code node IDs (from Superman)

─�� Autonomy (from Agentic Patterns) ──
autonomy_level string full|supervised|gated (HITL/HOTL/HOOTL)
agent_role string who handles this (architect|implementer|researcher...)
estimated_hours float task sizing (2hr sweet spot for agents)
